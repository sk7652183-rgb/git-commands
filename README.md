## 📊 Git Workflow Diagram

```mermaid
flowchart LR
    A[Untracked File] --> B[git add]
    B --> C[Staging Area]
    C --> D[git commit]
    D --> E[Local Repository]
    E --> F[git push]
    F --> G[GitHub Remote Repository]
```

⚙️ Example Workflow

# Create a file
echo "Hello Git" > hello.txt

# Stage changes
git add hello.txt

# Commit changes
git commit -m "Add hello file"

# Push to GitHub
git push origin master
